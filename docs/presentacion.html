La Solución

Sistema híbrido físico+digital que digitaliza todo el ciclo de vida del estacionamiento.

✓

Tarjetas Físicas con QR

Cada conductor recibe una tarjeta con código QR único al asignar espacio. Validación one-time en base de datos.

✓

PWA Multirrol

Una sola app web progresiva. 7 roles con UI filtrada por RLS. Sin instalar, sin app stores.

✓

Tiempo Real

Mapa interactivo de 110 espacios actualizado en <500ms vía WebSocket. Ocupación visible al segundo.

✓

Canal de Comunicación

Seguridad

QR One-Time

Validación atómica en base de datos. Cada QR usable una sola vez.

tarjeta.sql 
BEGIN;
UPDATE tarjetas SET activa = false
WHERE codigo = p_codigo_qr AND activa = true
RETURNING id, espacio_id;
-- 0 filas → QR ya usado o inválido
COMMIT;

🔒 Seguro

UUID único por tarjeta. Imposible de adivinar o reusar. DB rechaza segundo escaneo.

⚡ Rápido

Una query atómica. Sin locks ni race conditions. Validación en <100ms.
