07

QR One-Time

Validación atómica en base de datos. Cada QR usable una sola vez.

BEGIN;
UPDATE tarjetas SET activa = false
WHERE codigo = p_codigo_qr AND activa = true
RETURNING id, espacio_id;
-- 0 filas → QR ya usado
COMMIT;

Seguro

UUID único por tarjeta. Imposible de adivinar o reusar. DB rechaza segundo escaneo.

Rápido

Una query atómica. Sin locks ni race conditions. Validación en <100ms.
